## Featured Project

### AI Resume ↔ Job Description Matcher (Deployed Web App)

A production-style NLP system that evaluates resume alignment with job descriptions and identifies skill gaps.

**What it does**
- Accepts resume text or PDF upload
- Processes text using NLP pipeline
- Computes semantic similarity using TF-IDF + cosine similarity
- Generates match score and keyword gap analysis
- Provides real-time results via web interface

🔗 Live App: https://resume-matcher-webapp.onrender.com  
💻 Source Code: https://github.com/sreesushma/resume-matcher-webapp

**Tech Stack**
Python • Flask • scikit-learn • NLP • HTML/CSS • Cloud Deployment
